get /api/offloads
Gets list of topBoats offload (apiOffloadsGet)
By passing appropriet query parameters you can filter your result for your needs example request : https://fangsdata-api.herokuapp.com/api/offloads?fishingGear=%27Teiner%27,%27Juksa/pilkAndre%27&boatLength=%2715m%20-%2020,99m%27&

Query parameters

count (optional)
Query Parameter — Filters how many results you whant
fishingGear (optional)
Query Parameter — Filters of what fishing gear you are looking for.
boatLength (optional)
Query Parameter — Filters what boat lenth you are looking for. Two numbers From and to.
landingTown (optional)
Query Parameter — Filters what town the landings took place
landingState (optional)
Query Parameter — Filters what state the landings took place
fishType (optional)
Query Parameter — Filter by fish
month (optional)
Query Parameter — Filters after what month you are looking at. from and to.
year (optional)
Query Parameter — Filter after year you are looking at. From and to

Return type

array[TopOffloads]
